# JavaScript 继承

JavaScript 的继承的原理就是用原型链。另外函数只不过是在特定环境中执行代码的对象。

在 JavaScript 中，函数是对象的方法。如果一个函数不是 JavaScript 对象的方法，那么它就是全局对象的函数。

---

call() 和 apply() 之间的区别
不同之处是：

- call() 方法分别接受参数。
- apply() 方法接受数组形式的参数。

如果要使用数组而不是参数列表，则 apply() 方法非常方便。

- call/apply：fun执行的结果
- bind：返回 fun 的拷贝，并拥有指定的 this 值和初始参数
